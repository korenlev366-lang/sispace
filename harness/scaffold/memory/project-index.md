# Project Memory Index

Local harness memory for this checkout. Seeded by `sh harness/scripts/harness-bootstrap.sh`.

## Persistence

- Memory ledgers: `harness/memory/` (gitignored)
- Reports: `harness/reports/` (gitignored)
- Obsidian vault mirror: written to your Obsidian vault at sync time (not stored in git)

Set `SISPACE_HOME` to relocate the canonical memory store.
