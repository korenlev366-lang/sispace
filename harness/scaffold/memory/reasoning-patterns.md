# Reasoning Patterns

Durable reasoning patterns distilled from completed sessions. The post-task chain appends entries when reflection output includes pattern fields or extractable bullets.

## Required fields (each entry)

- **Problem type:** What class of situation triggered the friction or mistake
- **What worked:** Reasoning approach or fix that resolved it
- **What failed:** Approach that failed and why
- **When to apply:** Triggers for reusing this pattern in future sessions

```markdown
### PATTERN-YYYYMMDD-HHMMSS: [Short title]

- Session ID:
- Problem type:
- What worked:
- What failed:
- When to apply:
- Verification evidence:
```
